# apex-project-templates

Canonical project documentation templates for APEX-managed projects.

## Templates
- `templates/product-tool/` — For offline tools and digital products (e.g. VoltForge)
- `templates/content-engine/` — For automated content sites (e.g. GearGameDaily)
- `templates/saas/` — For subscription SaaS apps (e.g. DIY Cut List)
- `templates/governed-system/` — For high-governance systems (e.g. BATCH)

## How to Use
When APEX onboards a new project, it copies the appropriate template into the project repo and populates fields from the APEX database.
